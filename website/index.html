The problem

5 Packages. 22 KB. Zero Type Safety.

Most responsive stacks are still assembled from a bag of unrelated hooks, brittle viewport listeners, and hand-rolled CSS helpers. That means duplicate subscriptions, hydration bugs, and responsive logic scattered across your app.

One package handles media queries. Another does viewport width. A third pretends to do fluid typography. None agree on SSR.
You lose literal breakpoint inference, ship overlapping logic, and still end up guarding everything with typeof window.
fluidity-ts unifies breakpoints, clamp generation, container queries, prefers-* APIs, and SSR-first rendering into one coherent toolkit.

Use cases

Built for the places responsive code gets hardest.

SSR Apps

Resolve the best-fit breakpoint on the server, hydrate into the same layout, and stop shipping layout flicker to users.

import { headers } from 'next/headers'
import { resolveBreakpointFromHints } from 'fluidity-ts/server'

const { width } = resolveBreakpointFromHints(await headers())
return <ResponsiveProvider serverWidth={width}>...</ResponsiveProvider>

Design Systems

Create a single source of truth for tokens, breakpoints, fluid scales, and accessible motion preferences.

import { fluidScale, responsiveStyle } from 'fluidity-ts/styles'

export const type = fluidScale([
  'body', 'title', 'display'
], { minPx: 16, maxPx: 72 })

export const stack = responsiveStyle(system, 'gap', { xs: '1rem', lg: '2rem' })

Component Libraries

Ship primitives that adapt to their containers and remain type-safe across React, Vue, Svelte, and Lit wrappers.

import { containerQuery, defineContainer } from 'fluidity-ts/styles'

export const cardRoot = {
  ...defineContainer('card'),
  ...containerQuery('card', { minWidth: 420 }, {
    gridTemplateColumns: '1.4fr 1fr'
  })
}

Dashboards

Let data-heavy interfaces collapse, reorder, and simplify by breakpoint without hand-maintaining resize listeners.

const columns = useResponsiveValue({ xs: 1, md: 2, xl: 4 })
const bp = useBreakpoint()

<DataGrid
  dense={bp.below('md')}
  columns={columns}
  showSidebar={bp.above('lg')}
/>
